# Tirupur Vijay Cabs

Static GitHub Pages website for Tirupur Vijay Cabs.

## Files

- `index.html` - full single-page website
- `styles.css` - responsive styling
- `script.js` - mobile menu, trip tabs, and WhatsApp booking
- `.nojekyll` - lets GitHub Pages serve the site exactly as-is

## Preview locally

Open `index.html` in a browser, or run a simple local server:

```powershell
python -m http.server 8080
```

Then visit `http://localhost:8080`.

## Publish on GitHub Pages

1. Push this repository to GitHub.
2. Open the repository settings.
3. Go to **Pages**.
4. Choose **Deploy from a branch**.
5. Select the branch and `/ (root)` folder.
6. Save and wait for GitHub to publish the site.
